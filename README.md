# dcns-contract

![CI](https://github.com/dcnsdomains/dcns-contract/workflows/CI/badge.svg)

For documentation of the DcNS system

# How to setup

```
git clone https://github.com/dcnsdomains/dcns-contract
cd dcns-contract
npm install
```

# How to run tests

```
npx hardhat test
```
