# dcns-contracts

![CI](https://github.com/dcnsdomains/dcns-contracts/workflows/CI/badge.svg)

For documentation of the DcNS system

# How to setup

```
git clone https://github.com/dcnsdomains/dcns-contracts
cd dcns-contracts
npm install
```

# How to run tests

```
npx hardhat test
```
